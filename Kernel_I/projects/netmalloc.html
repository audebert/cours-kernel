NetMalloc

The aim of this project, is to develop a system call under Linux :

void *netmalloc(unsigned long size);

This one will create a specific memory region in the calling process.

That memory will be writeable and readable (and executable by default),
like a conventional malloced buffer, excepting that, its address and size
will be sent (sockets) to a distant machine, running a NetMalloc Server
that you will implement (userland and langage choice is free) and which
adress:port will be a paremeter of the module loading.

Usage example (just an example) :

$root@storage>netmalloc_server -l 192.168.0.3 -p 12345 &

$root@l33t>insmod netmalloc.ko "server=192.168.0.3:12345"
$root@l33t>cat netmallocer.c

#include <netmalloc.h>

int main()
{
    char *buffer = netmalloc(5120);
    memset(buffer, "A", 5120);
    write(buffer + 4090, 512);
    return 0;
}
